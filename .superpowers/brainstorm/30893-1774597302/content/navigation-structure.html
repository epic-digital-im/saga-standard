Tab Screens Detail

💬 Messages Tab
MessagesList
 ├── ConversationView (DM thread)
 │    └── MessageDetails
 ├── GroupList
 │    └── GroupConversation
 │         └── GroupInfo / Members
 ├── MemorySyncStatus
 └── NewMessage (compose)

• Real-time via saga-client-rt relay
• Encrypted envelopes (mutual/group scope)
• Offline mailbox drain on reconnect
• Memory envelope sync
📁 Documents Tab
DocumentExplorer
 ├── DocumentDetail
 │    ├── LayerViewer (9 layers)
 │    ├── EncryptionStatus
 │    └── TransferHistory
 ├── ExportFlow
 │    └── ContainerPreview
 ├── ImportFlow
 │    └── ValidationResults
 └── TransfersList
      └── TransferDetail

• Browse stored SAGA documents
• View encryption layers & status
• Export/import ZIP containers
• Initiate hub-to-hub transfers
🔍 Directory Tab
DirectorySearch
 ├── AgentProfile
 │    ├── NFT Details
 │    ├── TBA Address
 │    ├── Hub Status
 │    └── Send Message (→ modal)
 ├── OrgProfile
 │    ├── Members List
 │    ├── Governance Policy
 │    └── Join / Request Access
 ├── DirectoryProfile (Phase 7)
 │    └── Federation Status
 └── HandleResolver

• Search agents, orgs, directories
• View on-chain identity details
• Resolve handles → entities
• Initiate contact from profile
💰 Wallet Tab
WalletOverview
 ├── WalletDetail
 │    ├── TransactionHistory
 │    ├── TokenBalances (ETH, USDC)
 │    └── TBA Assets
 ├── SendFlow (→ modal)
 ├── ReceiveFlow (QR + address)
 ├── SubscriptionManager
 │    ├── ActivePlans
 │    ├── PaymentHistory
 │    └── RenewFlow (→ x402)
 └── WalletSettings
      ├── ManagedWalletConfig
      └── SelfCustodyBackup

• Dual-track: self-custody + managed
• Token balances & tx history
• x402 subscription management
• QR code receive addresses
👤 Profile / Me Tab
MyProfile
 ├── IdentityManager
 │    ├── AgentIdentities (owned NFTs)
 │    ├── OrgMemberships
 │    ├── DirectoryMemberships
 │    └── MintNew (→ modal)
 ├── HandleManager
 │    ├── Registered Handles
 │    └── UpdateHomeHub
 ├── SecuritySettings
 │    ├── BiometricSetup
 │    ├── PIN / Password
 │    ├── Backup & Recovery
 │    └── KeyManagement (KeyRing)
 ├── HubConnections
 │    ├── Connected Hubs
 │    ├── Relay Status
 │    └── AddHub (connect to new directory hub)
 ├── GovernanceAdmin (org admins only)
 │    ├── PolicyEditor
 │    ├── AuditLog
 │    └── MemberManagement
 └── AppSettings
      ├── Theme / Appearance
      ├── Notifications
      └── Network (chain selection)
